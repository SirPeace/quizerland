# Quizerland

🧩 The application to create, share and pass quizes!

⚛️ Developed using:

- _React_
- _Redux_
- _TypeScript_
- _Apollo (GraphQL)_
- _MaterialUI_

## Running locally

First, run the server using instructions from [here](https://github.com/SirPeace/quizerland-server).

🚀 Then, run the React app with these commands:

```
$ npm install
$ npm run generate
$ npm run serve
```

🎉 Enjoy!
